# ✅ Todo App

A modern Todo application built using React.

## 🚀 Features
- Add and delete tasks
- Mark tasks as complete
- Component-based architecture

## 🛠️ Technologies Used
- React
- JavaScript (ES6+)
- CSS

## 📦 Installation

### 1. Clone the repository
### 2. Install dependencies:

```bash
   npm install
```

### 3. Start development server:

```bash
   npm run dev
```

## 🌐 Usage
After running, open:
http://localhost:5173
